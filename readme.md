# 🌊 Surf Portfolio

홍보 및 콘텐츠 기획 작업을 정리한 개인 포트폴리오 사이트입니다.

---

## 📌 소개

브랜드의 성장을 돕는 콘텐츠와 캠페인을 기획하고 실행한 경험을 정리했습니다.
각 프로젝트는 문제 → 전략 → 실행 → 결과 구조로 구성되어 있습니다.

---

## 🚀 주요 기능

* 프로젝트 카드 형태 UI
* 반응형 웹 (모바일 지원)
* 스크롤 애니메이션
* 프로젝트 상세 페이지 제공

---

## 🛠 사용 기술

* HTML5
* CSS3 (반응형, 애니메이션)
* JavaScript (Intersection Observer)
